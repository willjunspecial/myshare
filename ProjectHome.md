# 翻墙软件 #

翻墙软件，可以越过网络防火墙的限制，从而浏览一些被屏蔽的网站。翻墙软件，又称“破网软件”、“破墙软件”和“穿墙软件”，是可以突破互联网审查封锁的东东。由于网络环境各不相同，不同的软件表现也各不相同，不一定都能用。系统不干净会影响翻墙，比如讨厌的百度地址栏搜索插件，会影响翻墙，用windows清理助手可以强制卸载掉，另外，换谷歌浏览器可以解决IE有时无法翻墙的问题。
